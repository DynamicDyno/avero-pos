## Avero POS

This is a React / Redux restaurant POS app.

### Installation

```
git clone git@github.com:DynamicDyno/avero-pos.git
cd avero-pos
yarn
yarn start
```

It should open up a tab at `localhost:3000` for you. (npm install should also work if you don't have yarn)
